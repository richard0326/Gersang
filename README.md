제가 cpp을 사용 안한지가 오래돼서 예전에 프로카데미 학원에서 공부했던 것을 토대로 다시 만들어봤습니다.

그래서 다시 공부할 겸해서 이것 저것 다시 만들어봤습니다.

그리고 기존의 장점을 어떻게 유지했는지도 적혀 있습니다.
제가 3~4년 경력을 쌓아가면서 나름 느낀 것을 토대로 재해석해 보았습니다.
